This demo shows SDK integration with user context.
The chatbot receives information about the user and their pet!

Integration Code

Pass context data before loading the SDK:

<!-- Step 1: Configure context (optional) --> <script> window.VetChatbotConfig = { userId: "user_123", userName: "John Doe", petName: "Buddy", source: "marketing-website", apiUrl: "http://localhost:3001" }; </script> <!-- Step 2: Load the SDK --> <script src="http://localhost:3001/sdk/chatbot.js"></script>

Context Fields

Field Description Required
userId Unique identifier for the user No
userName Name of the pet owner No
petName Name of the pet No
source Where the user came from No
apiUrl Backend API URL Yes (for production)
